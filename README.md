# Skin Consultation Management System
A java application developed for the manager in a consulatation center also that facilitate to add consultation for the patients.
## Facilities ##
### Manager ###
- Add a doctor (Maximum 10)
- Delete a doctor
- Print list of doctors
- Save/write in a file
### Patient (Using GUI) ###
- Display list of doctors
- Book a consultation with a selected doctor
- Auto assign a doctor if the selected one is unavailable at the time
- Adding notes and images about the consultation
- Visualization of details of a booked consultation
